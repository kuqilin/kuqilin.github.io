各站详解

🏞️
雷州西湖
千年经典园林，湖光山色、亭台错落。环境清幽静谧，适合散步放空、拍照打卡。
免费散步拍照1h
🏛️
雷州博物馆
城市人文缩影，馆藏丰富。特色石狗非遗文化——各种形态的石头狗，小孩看了特别有趣。
免费看石狗1h周一闭馆
🗼
三元塔
明代九层古塔，登顶可俯瞰整座雷州古城。公园免费进，登塔收费。古风氛围感十足，出片率极高。
公园免费登塔收费1h
🏯
雷祖祠
雷州核心文化地标，千年古建林立。了解雷州历史的第一站，中年长辈必定喜欢。
人文底蕴祈福1h
🏮
茂德公鼓城
古风网红商业街，白墙黛瓦。白天静谧拍照，夜晚灯火璀璨。吃喝玩乐一站式配齐，全家逛夜市。
免费夜市美食集中
